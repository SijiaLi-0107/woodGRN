Technical Features

• Gene Selection Pool: The selection pool dynamically filters out genes lacking any active regulatory links in the network database.

• Orthology Filtering: Nodes lacking valid orthologs in the selected species are automatically omitted.

• Scale Limitations: Graphic visualization is limited to 201 nodes and 200 edges, while the web table displays up to 200 rows. Use the export tool to retrieve the complete data list.
